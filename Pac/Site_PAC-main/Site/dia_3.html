Lista de Exercícios Básicos

1- LED Piscante: Desafio para iniciantes, onde o objetivo é criar um circuito que faça um LED piscar em um intervalo de tempo definido.
2- Sensor de Temperatura: Desafio para medir a temperatura ambiente utilizando um sensor e exibir o resultado em um display de LCD.
3- Semáforo: Desafio para criar um semáforo utilizando LEDs e programação para alternar entre as cores.
4- Controle de Motor DC: Desafio para controlar a velocidade e direção de um motor DC utilizando um driver de motor e um joystick.
5- Pisca-pisca: crie um circuito simples que faça com que um LED pisque de forma intermitente.
